# Mini MIPS Emulator
Created in C for the Computer Architecture subject.
 - Dynamic memory allocation
 - No global variables
 - Pointers
## Instruction examples
 - add $s0, $s1, $s2
 - sub $s0, $s1, $s2
 - mul $s0, $s1, $s2
 - div $s0, $s1, $s2
 - addi $s0, $s1, 100
 - li $s0, 100
 - j main
